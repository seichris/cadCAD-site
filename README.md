# Next.js TypeScript Tailwind ESLint Jest Prettier Template

Based on [this](https://github.com/paulintrognon/next-typescript) template

## Project setup and usage

Install dependencies:

```
yarn install
```

Run development server:

```
yarn dev
```

Build:

```
yarn run build
```

Run ESLint:

```
yarn lint
```

Format:

```
yarn format
```

Test:

```
yarn test
```
